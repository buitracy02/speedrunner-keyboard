# Sweaty SpeedRunner Keyboard

AI was used to help generate code. All content was reviewed by me.

made on a random thought if that I was typing it would play the audio of the sweaty speedrunner meme and wanted to try to make a python program with AI prompts

## **HOW TO USE:**

download the two files `loudkeyboard.py` & `loud_keyboard.mp3`

open program through terminal

`python loudkeyboard.py` to start program

**HOW IT WORKS**
* typing, plays audio
* stop typing, stops audio

`ctrl + c` to stop program or just close the terminal
